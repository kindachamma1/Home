Recent Achievements

Highlights of my latest accomplishments and projects.

1) Erasmus Mundus Program Selection

I was selected for the Erasmus Mundus program, ranking among the top 3% of students. This achievement granted me the EMSSE Top Applicant Scholarship with full tuition fee waiver.

2) French Robotics Championship 2023–2024

As part of the Robot Project, I was a member of an 8-person team competing in the Coupe de France de Robotique 2024. We built a main robot equipped with grippers to navigate the playground, carry plants, and deposit them precisely according to competition rules.

We also created two smaller robots called PAMIs, which were launched after the main robot's run to pollinate plants in designated zones.

My tasks included developing code for gripper movements, using the xBee module to communicate with PAMIs, and programming their movements and strategies. We worked with Arduino Uno, Mega, Nano boards, the xBee module, and an OrangePi board.

Obstacle avoidance was implemented using Ultrasonic Sensor HC-SR04 and TOF sensors.

Skills: Project Management · Time Management · Python · Robot Programming
